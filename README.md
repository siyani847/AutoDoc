# AutoDoc AI — MERN Stack
AI-Powered Code Documentation & Analysis Tool
Engineered an intelligent documentation system that automates code understanding and reduces onboarding time for developers.

---



## Stack

| Layer    | Tech                                  |
|----------|---------------------------------------|
| Frontend | React 18 + Vite                       |
| Backend  | Node.js + Express                     |
| Database | MongoDB + Mongoose                    |
| AI       | Google Gemini API|
| GitHub   | GitHub REST API v3                    |

---

## Architecture

User connects a GitHub repository
Backend fetches repository data via GitHub API
Files are processed and structured
AI model analyzes code and generates documentation
Frontend displays generated README and insights
